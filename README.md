This project analyzes consumer perceptions, awareness, and purchasing behavior toward eco-friendly and sustainable products using machine learning and data visualization techniques. 
The goal is to bridge the gap between environmental concern and actual purchasing behavior by providing data-driven insights and intelligent predictions.

Objectives:
Analyze consumer awareness and perception of eco-friendly products
Study willingness to pay for sustainable alternatives
Examine demographic influence on purchasing behavior
Predict eco-friendly purchase behavior using machine learning
Lay the foundation for an AI-based recommendation system
